-- INSTRUCTIONS D'INSTALLATION --

1. Installer Env3D sur votre ordinateur
2. Placer le dossier tux (models/tux/) dans le dossier models de votre projet Env3D
3. Placer le dossier sounds à la racine de votre projet Env3D
4. Placer le dossier src à la racine de votre projet Env3D
5. Placer le dossier data à la racine de votre projet Env3D
6. La méthode main se trouve dans le fichier Main.java du package main
7. Suivez les instructions dans la console
